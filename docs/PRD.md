# Claude Code 기능 소개 웹페이지 MVP PRD

## 핵심 정보

**목적**: Claude Code의 주요 기능 8가지를 카드 형태로 소개하고, 각 기능의 세부 내용을 상세 페이지에서 설명하는 공개 웹페이지
**사용자**: Claude Code에 관심 있는 개발자 및 잠재 사용자

## 사용자 여정

1. 메인 페이지 (기능 카드 그리드) → 8개 기능 카드 확인
2. 기능 카드 클릭 → 기능 상세 페이지 (세부 설명, 사용 예시)
3. "홈으로 돌아가기" 버튼 클릭 → 메인 페이지 복귀

## 기능 명세

| ID | 기능명 | 설명 | 관련 페이지 |
|----|--------|------|------------|
| F001 | 기능 카드 그리드 | 8개 Claude Code 기능을 카드 형태로 나열 | 메인 페이지 |
| F002 | 기능 상세 보기 | 각 기능의 세부 설명, 사용 예시, 주요 특징 표시 | 기능 상세 페이지 |
| F003 | 페이지 간 네비게이션 | 카드 클릭으로 상세 페이지 이동, 홈으로 돌아가기 | 메인 + 상세 페이지 |
| F004 | 다크모드 토글 | 라이트/다크 테마 전환 | 전체 페이지 |

## 페이지 구조

```
/                        → 메인 페이지 (기능 카드 그리드)
/features/[slug]         → 기능 상세 페이지
```

### 메인 페이지 (`/`)

- 히어로 섹션: Claude Code 소개 타이틀 + 설명
- 8개 기능 카드 그리드 (아이콘, 기능명, 한줄 설명)
- 카드 클릭 → `/features/[slug]` 이동
- 다크모드 토글

### 기능 상세 페이지 (`/features/[slug]`)

- 기능명 + 아이콘 헤더
- 기능 상세 설명 (2~3단락)
- 주요 특징 목록
- 사용 예시/시나리오
- **"홈으로 돌아가기"** 버튼 (상단 + 하단)

## 소개 기능 데이터 (8개)

| slug | 기능명 | 아이콘 |
|------|--------|--------|
| ai-code-editing | AI 코드 편집 | Code |
| terminal-commands | 터미널 명령 실행 | Terminal |
| file-system | 파일 시스템 탐색 | FolderSearch |
| multi-agent | 멀티 에이전트 | Users |
| memory-system | 메모리 시스템 | Brain |
| mcp-integration | MCP 서버 통합 | Plug |
| code-review | 코드 리뷰 & 분석 | Search |
| slash-commands | 슬래시 커맨드 | Slash |

## 비기능 요구사항

- 인증: 불필요 (공개 페이지)
- 반응형: 모바일/태블릿/데스크톱 지원
- 다크모드: next-themes 기반
- 데이터: 정적 TypeScript 상수 파일 (DB 불필요)
- 렌더링: SSG (정적 사이트 생성)
